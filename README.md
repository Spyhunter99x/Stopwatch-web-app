# Stopwatch Web App ⏱️

A simple and responsive Stopwatch web application built using HTML, CSS, and JavaScript. It allows users to start, pause, reset, and track elapsed time accurately.

## Features
- Start the stopwatch
- Pause and resume timing
- Reset the stopwatch
- Clean and responsive user interface
- Lightweight and easy to use

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git & GitHub

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Spyhunter99x/stopwatch-web-app.git
   ```
2. Open the project in your code editor.
3. Run `stopWatch.html` in any web browser.
